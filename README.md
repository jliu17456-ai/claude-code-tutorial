# Claude Code 通关秘籍 🚀

一份**由浅入深、生动有趣**的 Claude Code 使用教程网页，带你从 **小白** 一路晋级到 **大神**，
特别覆盖**全部斜杠命令**的速查与玩法。

## ✨ 这是什么

一个零依赖、可直接打开的单页教程网站。精心设计的前端界面，包含：

- 🌱 **第一章 · 入门**：这是什么神器 / 五分钟安装 / 第一句对话
- 🚀 **第二章 · 进阶**：四大核心概念 / CLAUDE.md 项目大脑 / 高效工作流
- ⚡ **第三章 · 斜杠命令**：40+ 命令**可搜索/可筛选/点击复制**的速查表，含自定义命令教程
- 🧙 **第四章 · 大神之路**：钩子 Hooks / MCP / 子代理军团 / 大神十二诀
- 🎯 **附录**：键盘快捷键速查 / 常见问题

## 🎨 亮点

- 响应式设计，深色主题，丝滑的滚动动画与 Hero 终端打字效果
- 斜杠命令实时搜索 + 分类筛选 + 一键复制
- 侧边栏导航高亮（scrollspy）、滚动进度条、回到顶部
- 纯静态：HTML + CSS + 原生 JS，**无需任何构建步骤**

## 🖥️ 本地预览

直接用浏览器打开 `index.html` 即可。或起一个本地服务器：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📁 文件结构

```
index.html   # 页面结构与全部教程内容
styles.css   # 全部样式
script.js    # 交互逻辑 + 斜杠命令数据
```

> 命令以官方 `/help` 输出为准；不同版本可能略有差异。
