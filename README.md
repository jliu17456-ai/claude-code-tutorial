一个零依赖、可直接打开的单页教程网站。精心设计的前端界面，包含：

- 🌱 **第一章 · 入门**：这是什么神器 / 五分钟安装 / 第一句对话
- 🚀 **第二章 · 进阶**：四大核心概念 / CLAUDE.md 项目大脑 / 高效工作流
- ⚡ **第三章 · 斜杠命令**：80+ 命令**可搜索/可筛选/点击复制**的速查表（标注 内置/技能/工作流/已移除 与版本），含自定义命令教程，并澄清 `ultrathink` 关键词 vs `/effort` 命令
- 🧙 **第四章 · 大神之路**：钩子 Hooks / MCP / 子代理军团 / 大神十二诀
- 🎯 **附录**：键盘快捷键速查 / 常见问题

## 🎨 亮点

- **中英双语**：右上角 `中/EN` 一键切换全站（含 93 条命令描述与终端动画），记住你的选择
- 响应式设计，深色主题，丝滑的滚动动画与 Hero 终端打字效果

## 🖥️ 本地预览

直接用浏览器打开 `index.html` 即可。或起一个本地服务器：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 📁 文件结构

```
index.html   # 页面结构与全部教程内容（中文为默认，带 data-i18n 标记）
styles.css   # 全部样式
script.js    # 交互逻辑 + 斜杠命令数据（含中英描述）
i18n.js      # 英文译文字典 + 中英切换逻辑
```

> 命令以官方 `/help` 输出为准；不同版本可能略有差异。
